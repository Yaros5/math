# math - невеликий додаток з графічним інтерфейсом, який допоможе вам з матрицями

## Можливості:
- розрахунок визначника матриці
- знаходження оберненої матриці
- транспонування матриць
- вирішення лінійних рівнянь методом Крамера

## Як користуватись?
1) Запустити файл "Program-exe for Windows" або "Program-exe for Linux" (в залежності від вашої операційної системи)
2) Відкриється вікно з головним меню, де вам потрібно вибрати, якою функцією програми ви бажаєте скористатись
3) Ввести дані, потрібні для виконання програми
4) Отримати результат
5) Радіти :)
